Memahami Konsep UI dan UX

Diterbitkan 23 september 2023 Mengembangkan pernyataan masalah untuk mendefinisikan kebutuhan pengguna. Pada course ini akan lebih difokuskan pada perancangan user-interface, Human Centered Design, Metode Design Thinking, menciptakan user personal, point of view, panduan praktis membuat how might we statement, affinity diagram, pembuatan wireframe menggunakan figma, pembuatan high fidelity prototyping menggunakan figma, kosenp user research, menentukan usability metric dan testing. User Interface (UI) Designer tidak bisa dipisahkan dengan User Experience (UX) Designer. UI Designer memiliki tugas menentukan tampilan aplikasi dan/atau situs. Sementara UX Designer menentukan bagaimana suatu aplikasi dan/atau situs bisa beroperasi dengan mudah.

Cascading Style Sheet merupakan aturan untuk mengatur beberapa komponen dalam sebuah web sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemrograman.
